# Environments

This section contains documentation about the different deployment environments.

This section provides an overview of the different environments used in the Liara project, from local development to production. Understanding these setups is crucial for contributing to the project, debugging issues, and deploying updates.

---

### Key Topics

- **<Keyword to="/app/system-overview/environments/local-dev">Local Dev</Keyword>**: Instructions on how to set up and run the entire application stack on your local machine.

- **<Keyword to="/app/system-overview/environments/docker-compose">Docker Compose</Keyword>**: Details on running the application using containerization for a more isolated and consistent setup.

- **<Keyword to="/app/system-overview/environments/azure-deployment">Azure Deployment</Keyword>**: A description of the production environment hosted on Microsoft Azure.
