# Backend: Node.js Server Overview

The backend is a standard Node.js server using the Express framework. It centralizes API handling, business logic, authentication, and database access.

## Responsibilities

- API routing and request handling
- Authentication and authorization via middleware
- Orchestration of chat, memory, and enrichment flows
- Database access through a service layer (Supabase client)
- Integrations with external AI services
