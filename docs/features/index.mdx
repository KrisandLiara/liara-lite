## Core Features

### Memory System
- **Tag Explorer**: Visual tag cloud with multi-stage search capabilities
- **Semantic Search**: Vector-based similarity search with adjustable thresholds
- **Memory Management**: Create, edit, delete, and organize memories

### Import Pipeline
- **Token Estimation**: Real-time cost calculation across multiple AI models
- **Smart Selection**: Separate include/enrich workflows for optimal processing
- **Visual Progress**: Step-by-step workflow with completion indicators
- **Compact Layout**: Space-efficient 2x2 grid design

### Chat Interface
- **Real-time Conversation**: Seamless chat experience with AI assistant
- **Memory Integration**: Automatic memory recall and storage
- **Voice Input**: Speech-to-text capabilities for hands-free interaction

### System Management
- **Live Status Console**: Real-time logging and progress monitoring
- **Debug Tools**: Comprehensive debugging and monitoring capabilities
- **Theme System**: Unified design system with consistent styling
