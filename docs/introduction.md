# Welcome to the Liara System Overview

This documentation provides a comprehensive overview of the Liara application architecture, data flow, and core modules. Use the navigation sidebar to explore the different sections.

## Purpose

The goal of this system is to create an intelligent chat assistant that not only responds to user queries but also builds a persistent, long-term memory of conversations. This allows for more contextual, personalized, and insightful interactions over time.

## Core Technologies

- **Frontend:** React, Vite, TypeScript, TailwindCSS
- **Backend:** Node.js, Express
- **Database:** Supabase (PostgreSQL)
- **AI/Embeddings:** OpenAI
