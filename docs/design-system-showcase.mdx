### Centralized Theme
All colors are now managed through `/src/lib/theme/liara-theme.ts`:

```typescript
import { LiaraTheme, liaraClasses } from '@/lib/theme/liara-theme';

// Use predefined classes
<div className={liaraClasses.bgPrimary}>
  <span className={liaraClasses.textSecondary}>Content</span>
</div>

// Or access theme values directly
const theme = LiaraTheme.background.primary;
```

### Consistent Backgrounds
- **All pages** now use `bg-slate-900` (Liara primary background)
- **Cards and panels** use `bg-slate-800` 
- **Elevated elements** use `bg-slate-700`
