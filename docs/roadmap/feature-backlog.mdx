# Feature Backlog

This is a collection of high-level ideas, experiments, and potential features to be tackled when the time is right. Items here are not scheduled but serve as a source of inspiration for future development cycles.

-   <Icon name="Compass" className="text-sky-400" /> **Vector Constellation View**: An interactive 2D/3D map of memory clusters, visualized by topic, emotion, or time.

-   <Icon name="Users" className="text-sky-400" /> **Custom Liara Personas**: Allow the user to switch between different system prompt modes (e.g., counselor, motivator, task manager).

-   <Icon name="Play" className="text-sky-400" /> **Memory Playback Mode**: A feature to step through past sessions as a visual timeline, including chat, images, and reflections.

-   <Icon name="Book" className="text-sky-400" /> **External Source Memory Ingestion**: Import and process data from external sources like email, social media, or RSS feeds into the vector memory with automatic tagging.

-   <Icon name="Puzzle" className="text-sky-400" /> **Modular Tool Use**: Grant Liara the ability to invoke a wider range of tools, such as file summarizers, calendar APIs, advanced search engines, or even CLI commands.

-   <Icon name="BrainCircuit" className="text-sky-400" /> **Self-Evolving Agent**: An advanced capability for Liara to suggest improvements to her own system prompt or propose new features based on usage analytics.
