# Milestones

These are major version checkpoints that reflect architectural or feature-complete stages of development.

### Completed Milestones

| Version | Title                      | Summary                                                                                                |
| :------ | :------------------------- | :----------------------------------------------------------------------------------------------------- |
| **v0.1**  | Foundational Stack         | Supabase DB, memory schema, vector search, basic frontend layout, and chat with system prompt.          |
| **v0.3**  | Memory Browser             | Implemented Memory UI (tag, timeline, search), vector imports, and memory-based recall.                |
| **v0.5**  | Liara-as-Agent Alpha       | Local Docker CLI tools, backend middleware auth, Liara agent memory integration (recall toggle).       |
| **v0.7**  | Documentation Portal       | Fully structured knowledge base with dynamic Markdown, Mermaid diagrams, and internal links.           |
| **v1.0**  | Liara MVP                  | Semantic memory integrated with contextual chat, session history, AI prompting, and system overview.   |

### Future Milestones
